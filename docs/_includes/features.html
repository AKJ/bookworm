100% Accessible

Bookworm is specially designed for blind and visually-impaired individuals. It is accessible to be used with a screen reader, a braille display, or using its builtin text-to-speech functionality. It is usefull for sighted users as well, as it offers a fully rendered version of the book.

Fully Featured

Bookworm contains all the features you need for a modern desktop reading experience. You can add Bookmarks, highlight interesting paragraphs, perform OCR, use text-to-speech, perform advanced searches, and search Wikipedia.

Blazingly Fast

Bookworm is comparatively faster than off-the-shelf eBook readers. Its speed even surpasses those eBook readers which have been designed specially for blind and visually impaired users.

Developed By Users

Bookworm is being developed by users for users. Its development has been started because of the inadequacy of the existing solutions. It will thrive because its first users are the people who are developing it.

Completely Free

Bookworm is a free software. You can download, use, and upgrade the program free of charge. It is also free as in "free speech", the source code of Bookworm is published on github for anyone to learn from and improve.

Special Features

Bookworm allows you to perform OCR on scanned documents, with more than 100 languages supported by the OCR engine. Bookworm also support structural navigation (also known as single letter navigation).
